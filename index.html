Projeto Documentações Back-End!

Pegue o café e espere um pouco, estou desenvolvendo o projeto de documentações Text. O objetivo desse projeto é verificar o conhecimento da candidata em Algoritmos e API's REST, para isso, foram utilizadas as seguintes tecnologias:

- Docker

- Node

- MySQL

- Sequelize (ORM)

- Nessas telas, deve-se consumir os 3 métodos criados na API, e exibir seus respectivos resultados de acordo com o input do usuário.

- Método 1 :

Word Frequency

rota: documents/word-frequency

body: {word: string}

Você deve implementar uma função que retorna o número de ocorrências de uma palavra no documento.

+ A operação deve ser case insensitive.
